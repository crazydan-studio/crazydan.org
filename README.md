Crazydan 站点源码
===================================

本项目基于 [Elm](https://elm-lang.org/) 编码并构建，并已实现：
- 语言国际化切换：目前支持中文、英文
- 页面自适应
- 初步的 Elm I18n 库设计实现
- ...

## 构建

```bash
bash ./build.sh
```

Note：构建产物存放在 `dist` 目录。

## 演示

```bash
elm reactor
```

浏览器打开 [http://localhost:8000/dist/index.html](http://localhost:8000/dist/index.html) 即可。
